### 注意事项

#### 电源
如果家有苹果的五福一安电源就够了，再淘宝买个usb转公线就可以了

#### 电阻
淘宝上买一袋袋的电阻，里面个种阻值都有

#### 电容
这个买103，104，105的电容就好，分别代表0.01µF ，0.1µF ，1µF 

#### 电线相关
买个剪线钳

#### 芯片
上面芯片的数量是搭建cpu用到的芯片数量，如果是那种只有一个的芯片，最好买多个，这样你可以在有问题的时候，能够确定是不是芯片的问题

## 视频
